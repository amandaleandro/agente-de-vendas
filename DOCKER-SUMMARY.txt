Automação:
  Linux/Mac (cron):
    0 2 * * * cd /path/agente-de-vendas && ./scripts/backup-db.sh

  Windows (Task Scheduler):
    Register-ScheduledTask -TaskName "FezinhaBackup" ...
    (veja DOCKER-SETUP.md para detalhes)

Restauração:
  gunzip -c backups/backup_*.sql.gz | docker exec -i fezinha-db psql ...
