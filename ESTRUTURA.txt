╔════════════════════════════════════════════════════════════════════════════════╗
║                            COMO INICIAR O BOT                                  ║
╚════════════════════════════════════════════════════════════════════════════════╝

1️⃣  CONFIGURAR VARIÁVEIS:
   cp .env.example backend/config/.env
   # Editar backend/config/.env com suas credenciais

2️⃣  INICIAR LOCALMENTE (Node.js):
   ./start.sh                    # Linux/Mac
   start.bat                     # Windows

3️⃣  INICIAR COM DOCKER:
   cd backend
   docker-compose up -d
   docker-compose logs -f

4️⃣  ACESSAR PAINEL:
   http://localhost:3099
